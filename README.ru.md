# RankWise - Профессиональный SEO-движок

\[ [English](README.md) | **Русский** \]

> **Ранжируйтесь выше: пишите для людей в первую очередь, для поисковиков - во вторую.**

AI-скилл для генерации, рерайта и аудита контента по 49 факторам ранжирования. Полное покрытие всех ключевых SEO-сигналов: schema-разметка, E-E-A-T, оптимизация под featured snippet.

---

## Что это?

**RankWise** - это системный промпт (скилл) для любой LLM: GPT, Claude, Gemini, DeepSeek. Он превращает ИИ в SEO-стратега мирового уровня, который:

- Пишет контент, набирающий 46+/49 по чек-листу факторов ранжирования
- Переписывает существующий контент, закрывая SEO-пробелы (из URL или текста)
- Аудирует контент по 49 измеримым сигналам ранжирования
- Генерирует оптимизированные мета-теги, alt-тексты и URL-slug'и
- Создаёт полные SEO-брифы и редакционные планы

Покрывает все ключевые SEO-сигналы: читаемость, schema-разметка, E-E-A-T, featured snippet, тематические кластеры, LSI-стратегия.

---

## Как работает

1. **Загрузите `SKILL.md`** как системный промпт в любую LLM.
2. **Дайте задачу:** «SEO статья. Ключ: email маркетинг.» / «Напиши о email стратегии + мета. Ключ: email.» / «Аудит этого контента.» / «Полный пакет для лендинга.»
3. **Получите результат** - только то, что запросили. Без лишних модулей. Без воды.

**Шесть режимов с модульной выдачей:**

- **Generate** - SEO-контент с нуля (ключ → статья, тема + ключ → статья + мета)
- **Rewrite** - Рерайт (URL или текст) с SEO-оптимизацией и удалением AI-маркеров
- **Audit** - Оценка по 49 факторам с взвешенным скорингом, грейдом, приоритетными правками
- **Meta-Only** - Мета-теги, описания, OG, alt-тексты, URL
- **Brief** - Контент-бриф: ключи, структура, план ссылок, медиа-план
- **Quick-Fix** - Точечные правки: «исправь длину title», «добавь ключ в H2», «уменьши пассивный залог»

**Контекстные умолчания:**
- Только ключ → статья (без меты и alt'ов)
- Тема + ключ → статья + мета
- Тема + ключ + картинки → полный пакет (статья + мета + alt'ы)
- Любая комбинация - explicit: «только статья», «только мета», «полный пакет»

## 49-факторная SEO-матрица

Охватывает всё от размещения ключевых слов до schema-разметки:

| # | Фактор | Цель |
|---|--------|------|
| K1 | Фокусное ключевое слово задано | Должно быть |
| K2 | Ключ в SEO title | Да |
| K3 | Ключ в начале title | Первые 3-5 слов |
| K4 | Ключ в meta description | Первые 20 слов |
| K5 | Ключ в URL | Присутствует |
| K6 | Ключ в начале контента | Первые 150 слов |
| K7 | Ключ в теле контента | Естественно распределён |
| K8 | Ключ в подзаголовках | Минимум 1 H2 |
| K9 | Ключ в alt тексте | Минимум 1 изображение |
| K10 | Плотность ключа | 0.8%–1.5% |
| K11 | Проверка каннибализации | Нет дублей |
| K12 | Длина ключевого слова | 3–X символов |
| C1 | Количество слов | 600+ |
| C2 | Короткие абзацы | ≤3 предложения |
| C3 | Изображения/видео | ≥1 на 300 слов |
| C5 | Число в заголовке | ≥1 |
| C6 | Power words в заголовке | ≥2 |
| C8 | Читаемость | Grade 7–9 |
| C9 | Пассивный залог | ≤10% |
| C13 | Иерархия заголовков | Правильная H1→H2→H3 |
| L1 | Внутренние ссылки | 3–10 |
| L2 | Внешние ссылки | 2–5 |
| L9 | Title-атрибуты ссылок | Присутствуют |

*...полный чек-лист из 49 факторов в `shared/checklist.md`*

## Быстрый старт

**Только ключ → статья:**
> «SEO статья. Ключ: стратегия email-маркетинга.»

**Тема + ключ → статья + мета:**
> «Напиши о стратегии email-маркетинга для SaaS-фаундеров. Ключ: email стратегия. Язык: ru.»

**Полный пакет (статья + мета + alt'ы):**
> «Полный пакет о контент-маркетинге. Ключ: контент стратегия. Картинки: дашборд, график, фото.»

**Только мета-теги:**
> «Сгенерируй meta title и description для лендинга SaaS. Ключ: управление проектами.»

**Только alt-тексты:**
> «Сгенерируй SEO alt-тексты для 5 изображений. Ключ: email-маркетинг.»

**Полный аудит:**
> «Сделай SEO-аудит этого контента: [текст]. Ключ: управление проектами.»

**Контент-бриф:**
> «Создай SEO-бриф для pillar page о контент-маркетинге. Ключ: стратегия контент-маркетинга.»

**Рерайт из URL:**
> «Перепиши эту статью для SEO: [URL]. Ключ: SEO чек-лист.»

**Совместно с HumanAI:**
> «SEO-рерайт через RankWise, затем очеловечь через [HumanAI](https://github.com/MADEVAL/HumanAI). RU. Ключ: SEO-аудит.»

**Совместно с MindFluence:**
> «RankWise SEO-бриф для лендинга, затем [MindFluence](https://github.com/MADEVAL/MindFluence) из этого брифа. Тон: expert-calm.»

## Интеграция с HumanAI и MindFluence

RankWise спроектирован для совместной работы:

- **[HumanAI](https://github.com/MADEVAL/HumanAI)** - Text Humanization Engine. RankWise → структура SEO → HumanAI → человеческий голос. Следите за плотностью ключей после очеловечивания.
- **[MindFluence](https://github.com/MADEVAL/MindFluence)** - Cognitive Bias Marketing Engine. RankWise → SEO-структура → MindFluence → когнитивное убеждение. Следите за переспамом ключей в `bold-sell` тоне.
- **Тройная интеграция:** RankWise (структура) → MindFluence (убеждение) → HumanAI (голос) → RankWise Audit (финальная проверка).

Подробнее в `SKILL.md` → INTEGRATION WITH OTHER SKILLS.

## Требования

- Любая LLM с полем системного промпта
- Для полной функциональности: система скиллов с загрузкой файлов из папки (OpenCode, Claude Code)
- **SKILL.md самодостаточен** - содержит все ключевые правила для standalone-использования. `shared/` и `scenarios/` добавляют профессиональную глубину (формулы скоринга, списки AI-маркеров, читаемость по языкам, шаблоны schema, контент-типы)
- Без API-ключей, без инструментов, без зависимостей - чистый промпт-инжиниринг
- Работает на любом языке; языковые параметры в `shared/readability-params.md` и `shared/power-words.md` для 9 языков (EN, RU, UK, DE, FR, ES, PT, IT, PL)

## Установка

### OpenCode
```
rankwise/ → .opencode/skills/          (проект)
          → ~/.config/opencode/skills/  (глобально)
```

### Claude Code
```
rankwise/ → ~/.claude/skills/
```

### Любая LLM (standalone)
Скопируйте содержимое `SKILL.md` как системный промпт. Добавьте файлы из `shared/` и `scenarios/` для расширенного контекста.
